# Chest Cavity Beyond

一个基于 NeoForge 的 Minecraft 1.21.1 模组，扩展了胸腔器官系统的内容和玩法。

## 模组信息

- **Minecraft 版本**: 1.21.1
- **NeoForge 版本**: 21.1.216
- **模组版本**: 1.0.0
- **作者**: zhaiji

- **美术资源许可证**: Apache License 2.0（来自 [Chest Cavity](https://github.com/Tigereye504/chestcavity)）

## 简介

Chest Cavity Beyond 是一个独立的扩展模组，为游戏添加了丰富的器官类型和生物器官系统。玩家可以使用开胸器从各种生物身上获取器官，并通过器官替换获得特殊能力。

## 主要特性

### 丰富的器官类型

- **基础器官** - 心脏、脊柱、肺脏、胃、肠子、肾脏、脾脏、肝脏、阑尾、肋骨、肌肉
- **动物器官** - 陆生动物、小型动物的各种器官变体
- **水生器官** - 鳃、鱼类肌肉、鱼骨、鱼肠等
- **抗火器官** - 烈焰人器官系列，提供火焰抗性
- **末影器官** - 末影人器官系列，提供传送相关能力
- **运动肌肉** - 力量型、速度型、弹跳型肌肉
- **特殊消化器官** - 食肉动物胃、食草动物胃、食草动物瘤胃

### 生物器官系统

为多种 Minecraft 生物添加了独特的器官：

**常见生物**
- 猪、牛、羊等 - 标准动物器官
- 狼 - 食肉动物消化系统
- 羊、兔 - 食草动物瘤胃
- 兔子 - 弹跳型肌肉

**特殊生物**
- 苦力怕 - 苦力怕阑尾（可爆炸）、苦力怕叶
- 烈焰人 - 烈焰核心、活性烈焰棒（发射火球）
- 恶魂 - 恶魂胃、气囊（发射大型火球）
- 潜影贝 - 潜影贝脾脏（发射潜影子弹）
- 守卫者 - 守卫者之眼（激光攻击）
- 远古守卫者 - 远古守卫者之眼
- 末影龙 - 完整龙类器官系统（发射龙息弹）
- 凋零 - 凋零脊柱、肋骨、扭曲灵魂沙
- 铁傀儡 - 傀儡核心、傀儡电缆、活塞肌肉、装甲板
- 幽匿守卫 - 幽匿心脏、脊柱、核心（音爆攻击）

**节肢生物**
- 蜘蛛 - 丝腺
- 洞穴蜘蛛 - 毒腺
- 蝎蛉 - 节肢器官

### 器官属性系统

不同器官提供各种属性加成：
- 健康值、防御、力量、速度
- 消化效率、营养获取效率
- 新陈代谢效率、耐力
- 肺活量、呼吸效率、神经效率

### 特殊机制

- **胸腔系统** - 每个生物拥有 3×9 的器官格子
- **开胸手术** - 使用开胸器获取和替换器官
- **器官技能** - 特殊器官可触发独特能力
- **附魔系统** - 远程操作、先进手术、谨慎手术、安全手术
- **发射系统** - 安装特定器官可发射火球、子弹、激光等

### 新增配方

- **开胸器** - 杠杆 + 铁锭 + 木棍
- **史莱姆胃** - 史莱姆核心 + 胃 + 史莱姆球
- **岩浆胃** - 岩浆怪核心 + 胃 + 岩浆膏
- **腐肉** - 由腐烂器官合成
- **骨粉** - 由脊柱和肋骨制作
