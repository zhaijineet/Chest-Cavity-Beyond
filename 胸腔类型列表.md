# 胸腔类型列表

本文档列出了所有胸腔类型及其详细信息，包括器官配置和呼吸需求。

ChestCavityType是胸腔类型的结构
CommonEventHandler#handlerFMLLoadCompleteEvent管理实体类型和胸腔类型关联的注册
ChestCavityManager则是胸腔类型的注册

后续更改任务你只能从外部库net.minecraft.world.item.Items和本项目的InitItem里选择物品当作胸腔器官，不允许新建器官

且你更改了实体类型和胸腔类型关联的注册的注册时，需要调整他们的位置到正确的地方

完成更改任务后，更新列表数据，保留"更改任务，移除更改任务下面的任务列表

使用用中文交流

简写：
r为行
c为列
例：
1r2c 为1行2列
